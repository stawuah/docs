## Install

swiftui-rn requires React Native ≥ 0.71 and React ≥ 18.

<CodeGroup>

```bash npm
npm install swiftui-rn
```

```bash yarn
yarn add swiftui-rn
```

```bash pnpm
pnpm add swiftui-rn
```

</CodeGroup>

<Note>
`react` and `react-native` must already be installed in your project. swiftui-rn lists them as peer dependencies and does not bundle them.
</Note>

## Your first screen

Import primitives, build a node tree, pass it to `<Render>`.

```tsx App.tsx
import React from 'react'
import { VStack, Text, Button, Render } from 'swiftui-rn'

export default function App() {
const screen = VStack(
Text('Hello, swiftui-rn')
.title()
.color('#fff'),

Text('No JSX. No StyleSheet.')
.caption()
.muted(),

Button('Get Started', () => console.log('🚀'))
.bg('#16a34a')
.padding(16)
.rounded(12),
)
.bg('#0f172a')
.padding(24)
.gap(16)
.fill()
.center()

return <Render node={screen} />
}
```

That's it. Run it on your simulator — you'll see a dark screen, a title, a caption, and a green button.

## Step by step

<Steps>
<Step title="Import primitives">
Import the node factories you need from `swiftui-rn`. Common ones are `VStack`, `HStack`, `Text`, `Button`, and `Render`.

```tsx
import { VStack, HStack, Text, Button, Spacer, Render } from 'swiftui-rn'
```
</Step>

<Step title="Build a node tree">
Call factory functions and chain modifiers. Pass children as arguments to container nodes.

```tsx
const screen = VStack(
Text('Balance').caption().muted(),
Text('$1,240.00').title().color('#fff'),
Spacer(),
Button('Send', onPress).bg('#16a34a').padding(18).rounded(14).fullWidth(),
)
.bg('#0f172a')
.padding(24)
.gap(12)
.fill()
```
</Step>

<Step title="Render it">
Pass the root node to `<Render>`. One component renders the entire tree.

```tsx
return <Render node={screen} />
```
</Step>
</Steps>

## Dynamic screens with state

When your tree depends on state or props, wrap it in `useMemo()` so it only rebuilds when needed.

```tsx SendScreen.tsx
import React, { useState, useMemo } from 'react'
import { VStack, Text, Input, Button, Spacer, Render } from 'swiftui-rn'

export default function SendScreen() {
const [amount, setAmount] = useState('0')

const screen = useMemo(() =>
VStack(
Text('You send').caption().muted(),

Input('0.00')
.value(amount)
.onChange(setAmount)
.numeric()
.fontSize(48)
.bold()
.color('#fff')
.textAlign('center')
.bg('transparent'),

Text(`≈ GHS ${(+amount * 12.8).toFixed(2)}`)
.body()
.color('#16a34a')
.textAlign('center'),

Spacer(),

Button('Send Now', handleSend)
.bg('#16a34a')
.padding(18)
.rounded(16)
.fullWidth(),
)
.bg('#0f172a')
.fill()
.padding(24)
.gap(16),
[amount])

return <Render node={screen} />
}
```

<Tip>
Static node trees (no state dependency) can be defined **outside** the component entirely. This means zero rebuild cost on every re-render.
</Tip>
